# kittycracker

**Linear OTP & RNG Forensics Framework**

kittycracker is a research-focused framework for analyzing observed One-Time Password (OTP) sequences and determining whether they originate from **reversible linear generators**.

The project supports security research, legacy system analysis, and study of OTP generator design.
